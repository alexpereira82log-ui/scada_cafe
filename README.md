# scada_cafe
Gerenciamento de informações e análises Scada Café
Projeto para análise e direcionamento para ações que visam aumento de faturamento e eficiência operacional da loja Cinema (Scada Café).
Análises de Faturamento, histórico de vendas, ticket médio, número de cupons, perdas, eficiência de vendas por equipe, etc.
